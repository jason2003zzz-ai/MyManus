# Word 与 PDF 文档生成

用于把现有材料整理成带标题、段落、列表和表格的正式 Word 报告，也用于创建或修改 DOCX、PDF、调研总结、申请材料和其他可交付文档。关注内容结构、排版一致性、文件可打开性和最终渲染效果；仅以 Excel 工作簿为交付物时改用表格技能。

## 执行流程

1. 确认文档目的、读者、语言、篇幅、模板和必须保留的原文。
2. 先构建标题层级和内容提纲，再写入段落、列表、表格、页眉页脚及引用。
3. 使用项目提供的 Office 或 Python 文档工具生成文件，不把纯文本伪装成文档产物。
4. 生成后重新读取文件内容，核对标题、段落、表格、数字和链接。
5. 如支持渲染，检查分页、溢出、字体、留白和表格宽度；修复后再交付。

## 交付要求

- 返回文件的明确路径或下载链接。
- 说明文件类型、主要内容和验证结果。
- 不覆盖用户已有文件，除非用户明确要求。

# 联网检索与资料调研

用于查找最新资料、比较多方观点、核实事实、阅读网页或论文并形成带来源结论的研究任务。优先选择官方文档、论文、政府或机构页面等一手来源。

## 执行流程

1. 明确研究问题、时间范围、地区、比较维度和交付格式。
2. 设计多组互补查询词，先定位权威来源，再补充不同视角的独立来源。
3. 区分发布日期与事件发生日期；对可能变化的信息记录查询时间。
4. 对关键结论进行交叉验证，发现来源冲突时说明定义、口径或时间差异。
5. 输出时先给结论，再给证据、限制和来源链接；不得用搜索摘要替代原文核验。

## 质量要求

- 不编造引用、作者、数据和链接。
- 直接引语保持简短并准确标注来源。
- 无法验证的内容应标记为推测或待确认。

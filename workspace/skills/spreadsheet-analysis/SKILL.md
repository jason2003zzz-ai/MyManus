# Excel 表格处理与分析

用于读取、清洗、合并、计算和生成 Excel、CSV、XLSX 表格，也适用于公式、数据透视、条件格式和结构化报表任务。必须保证数据类型、计算口径和输出文件可复核。

## 执行流程

1. 检查工作表、列名、数据类型、缺失值、重复值、时间范围和单位。
2. 在计算前明确指标定义、分母、过滤条件和异常值处理方式。
3. 优先保留原始数据，新建结果工作表或输出新文件，避免不可逆覆盖。
4. 公式引用应覆盖完整数据范围；汇总值与明细随机抽样核对。
5. 输出后重新打开工作簿，验证工作表名称、公式、格式、冻结窗格和文件完整性。

## 质量要求

- 金额、百分比、日期和空值使用正确格式。
- 不把文本数字直接参与计算。
- 对推断或修复的数据留下说明。

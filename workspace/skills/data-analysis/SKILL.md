# Python 数据分析与可视化

用于使用 Python 完成数据清洗、探索性分析、统计计算、指标诊断、建模和图表生成。分析必须可复现，结论必须由数据结果支持。

## 执行流程

1. 读取数据后报告规模、字段、类型、缺失、重复和异常分布。
2. 明确分析问题、指标定义、分组维度、基准和假设。
3. 使用 pandas、NumPy 或统计库进行可复现计算，避免手工抄录中间结果。
4. 图表选择应匹配问题：趋势用折线，比较用条形，分布用直方或箱线，关系用散点。
5. 对关键数值进行独立复算或抽样检查，并区分相关性与因果关系。

## 交付要求

- 保存代码、清洗后的数据、表格或图表等产物。
- 先给核心发现，再给证据、限制和下一步建议。
- 样本不足或数据质量较差时不得给出过度确定的结论。
